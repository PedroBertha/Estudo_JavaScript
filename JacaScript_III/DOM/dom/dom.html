Estudando DOM

Documento Object Model, ou Modelo de Objeto de Documento, é uma interface de programação para documentos HTML e XML.

Pertimite que um documento escrito em linguagem de marcação (html, XML, etc) seja acessado pelo JavaScript.

Isso é feito através de uma estrutura hierárquica de objetos, estrutura de árvore de objetos HTML -> body -> h1, p, table -> tr -> td, etc

Pertmite que tenhanmosa acesso direto à página HTML para manipulá-la de várioas formas.

No navegador, podemos acessa o DOM atráves do obejto documen
